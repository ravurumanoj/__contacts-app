1-run the command (ng new contacts-app) in command prompt to create angular app
2-after creating the project open the folder through visual studio code (folder locaton:- c/users/username).
3-then create the different component names using the command (ng generate component component name).
component name-signin,signup,contactsform
4-in the component automatically it will create 4 files including html,css,typescript,spec.
5-then write the code for both html and css file.
6-in the app.routing.module we need to add the paths of all components which helps to redirect the page.
7-to run the project use (ng serve) command in vs code command prompt.
8-the server will create then this link will appear (http://localhost:4200/) then copy the link and
paste it in the chrome.
9-then we can see the web application.
